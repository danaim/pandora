SPiCT app a4a app

Through this application the user can easily implement two stock assessment models without no programming knowledge required. The two models are the stochastic surplus production model in continious time (SPiCT) and the statistical catch-at-age stock assesment model developed as part of the Assessment For ALL (a4a).For both models general information is provided about the dependencies, type of data needed and settings.
